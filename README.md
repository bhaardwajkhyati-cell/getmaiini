# GetMaiini — AI Spend Audit Tool

GetMaiini is a free web app that helps startups find where they're
overpaying on AI tools like Cursor, Claude, ChatGPT, and GitHub Copilot.
It's a lead generation tool built for Credex, a company that sells
discounted AI credits.
